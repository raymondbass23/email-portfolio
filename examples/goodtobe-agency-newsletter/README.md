# GoodToBe Agency Newsletter

## Technical Specifications

- **Type:** Marketing Newsletter
- **Responsive:** Yes (Mobile-first approach)
- **Client Compatibility:** Gmail, Outlook, Apple Mail, Yahoo
- **Testing:** Litmus/Email on Acid compatible

## Code Features

- HTML Tables for layout structure
- Inline CSS for maximum compatibility
- Media Queries for mobile responsiveness (max-width: 480px)
- Outlook conditional comments for specific styling
- Web-safe fonts with fallbacks
- Alt text for all images
- Semantic HTML structure

## Mobile Optimization

- Stacking columns on small screens
- Touch-friendly button sizes (min 44px)
- Readable font sizes (16px minimum)
- Optimized padding for mobile viewing

## Performance

- Minimal CSS in `<style>` block
- Efficient table structure
- Fast loading times
